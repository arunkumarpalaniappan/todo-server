### todo-server

A Simple to-do application with aws dynamo db database connectivity and user authentication.

#### Application Stack/Tech
- Heroku
- Node
- Hapi
- Express
- Blue Bird

#### Data Base
- AWS Dynamo DB

#### License

#### API End Points

`/login`

`METHOD: POST`

`parameters: username,password`

`RESPONSE: JWT Token or 401`
